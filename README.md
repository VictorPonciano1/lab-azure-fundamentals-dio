# ☁️ Laboratórios Azure Essentials (DIO)

> 🧑🏻 Autor: Victor Ponciano — https://github.com/VictorPonciano1  

Este repositório reúne **anotações, atividades práticas e resumos técnicos**
realizados durante o curso **Azure Essentials**, oferecido pela plataforma DIO.

O objetivo é documentar o aprendizado adquirido nos laboratórios, servindo
como material de consulta futura e também como registro da evolução nos
estudos sobre computação em nuvem utilizando Microsoft Azure.

---

## 🎯 Objetivos do Repositório

- Consolidar conhecimentos fundamentais sobre **Microsoft Azure**
- Registrar aprendizados obtidos nos laboratórios práticos
- Organizar resumos técnicos para revisão futura
- Servir como portfólio de estudos em Cloud Computing

---

## 📂 Conteúdo

Os diretórios deste repositório podem conter:

- 📝 Resumos dos laboratórios
- ⚙️ Configurações realizadas no Azure
- 📚 Conceitos importantes de computação em nuvem
- 💡 Observações e aprendizados pessoais

---

## ☁️ Tópicos abordados

Durante o curso são explorados conceitos como:

- Modelos de computação em nuvem (IaaS, PaaS e SaaS)
- Regiões e recursos do Azure
- Máquinas Virtuais
- Redes Virtuais
- Segurança e controle de acesso
- Custos e gerenciamento de recursos
