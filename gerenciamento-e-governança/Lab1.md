# LABORATÓRIO - GERENCIAMENTO DE CUSTOS

Calculadora de Custo Total de Propriedade (TCO): "estime a economia de custos gerada pela migração de cargas de trabalho para o Azure."

A TCO compara On-Premises vs Nuvem. Ela não calcula apenas preço de serviços Azure isolados.

Possui 3 fases:
1. Definir cargas de trabalho: cada uma possui suas próprias especificações para uma melhor presição.
    - Servidores;
    - Bancos de Dados
    - Armazenamento;
    - Redes;
    - Licenciamento;
    - Custos Operacionais;

>Cobertura de Software Assurance: permite reaproveitar licenças existentes (Azure Hybrid Benefit), reduzindo custos estimados na TCO.

2. Ajustar suposições.
    - Crescimento anual;
    - Tempo de Projeção;
    - Renovação de Hardware;

3. Exibir Relatórios.
    - Comparação Gráfica;
    - Economia Estimada;
    - Detalhamento por Categoria;
    - Projeção Futura;

O(s) relatório(s) mostram uma projeção gráfica de economia detalhada comparando com um ambiente On-Premise e o ambiente em nuvem.

A TCO faz cálculos que indicam o quanto é gasto hoje e quanto será gasto no futuro.

Além da TCO, temos também a Calculadora de Preços do Azure que é focada nos serviços/produtos Azure em si, como:
- VMs (Máquinas Virtuais);
- Armazenamento;
- Banco de Dados (SQL);
- Functions (Funções);
- Serviços de IA;

Entre inúmeros outros serviços/produtos que também estão disponíveis no Azure Marketplace.

A calculadora não compara com ambiente local. Ela calcula custo estimado apenas no Azure.

>**IMPORTANTE: Ambas as calculadoras fazem seus cálculos com base nas regiões selecionadas, pois o preço pode variar de acordo com cada região.**

Monitoramento/Gerenciamento de Custos é outra Ferramenta disponibilizada pela Azure que mostra o Overview completo para que seja possível observar aquilo que é gasto (valores).
